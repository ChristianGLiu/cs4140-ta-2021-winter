JavaScript CRUD Tutorials (for Dal Students)

 

DB Setup

Step 1: Setup CS ID  if you did not yet

Navigate to (https://csid.cs.dal.ca/), and make sure you have working CSID; Note down the username/password for the following steps.

Step 2: Setup Maria/MySql Database in DAL CS domain

Navigate to (https://web.cs.dal.ca/databases), and login with the username/password from Step 1; Click the “setup database” button, you should see the password for accessing database appearing on the top of the panel.

Keep in mind, the password generated is not easy to memorize. You might want to change the password. Then, please go to optional Step 3 to change password. Otherwise, note down the generated password for future programming.

Step 3 (Optional): Login phpMyAdmin to change the password (or for other data operations )

Navigate to (https://myadmin.cs.dal.ca/); Click “Change password” link in the red circle as shown in the screenshot; Follow the instruction to complete the change password operation.

Step 4 (Option 1): Create Database / Tables via phpMyAdmin portal

Continue to Step 3; System might already generate a Database named as your account name; Select “SQL” tab, input the SQL code to create tables required;

Step 4 (Option 2): Login Bluenose server; Create Database

Use SSH client tools to logon Bluenose server (YOUR_OWN_CSID@bluenose.cs.dal.ca), like the screenshot below:

Note, you could use any SSH clients to logon bluenose other than PUTTY.

Create Database connection via the command in the shell of Bluenose server:

mysql -u YOUR_OWN_CSID -p -h db.cs.dal.ca

Then, write your own SQL code to create tables. Please refer to sample SQL code in Appendix. The result should be like the screenshot with phpMyAdmin GUI:

 

Database CRUD programs (Java and JavaScript)

JavaScript:

Javascript database connection is implemented with nodejs extension. Students can obtain plenty of nodejs materials online, such as : https://www.w3schools.com/nodejs/nodejs_mysql.asp The Dal Bluenose server support nodejs, so students can upload their script file to Bluenose server for execution.

The sample executable programs are attached in Appendix. Here, we show what suggested results students should get:

Create Database Connection

Create(insert) New Record

Update a Record

Read a Record or Read All Records

Delete a Record

Java:

Dependency

mysql connection library

Students could download this JAR file from official MySQL website: https://dev.mysql.com. For instance, on Dal bluenose server, we can execute command:

wget http://www.java2s.com/Code/JarDownload/mysql/mysql-connector-java-5.1.23-bin.jar.zip

Then, unzip this file as executable JAR file

Once obtained this JAR file, students should include this JAR whenever compiling their Java classes, like:

java -cp .:mysql-connector-java-5.1.23-bin.jar JavaCRUDSampleCode
