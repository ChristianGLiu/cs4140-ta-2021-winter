Database CRUD programs (PHP)

Introduction

PHP is a working language on DAL Bluenose server, it means students do not extra configuration for the standalone application execution and web services implementation. There are plenty of tutorial materials available online, such as https://www.w3schools.com/php/. We also provide the sample CRUD code tailored for our course only. If students do not set up database yet, please students check the  Database Setup tutorial document.

Besides, the PHP code essentials can work for both standalone application and web services. Therefore, we can test it ether by command line:

php YOUR_PHP_APP.php  parameter1 parameter2….

Or through the web browsers. The path is based on the configuration of the apache2 server. For instance, the DAL Bluenose default web home is under /users/grad/YOUR_CSID/public_html .
